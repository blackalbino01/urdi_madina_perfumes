This folder should contain the following images for your perfume website:

## Required Images

1. **hero-bg.jpg** - Background image for the hero section (1920x1080 recommended)
2. **perfume1.jpg** - Product image for "Classic Elegance" (400x400 recommended)
3. **perfume2.jpg** - Product image for "Modern Muse" (400x400 recommended)
4. **perfume3.jpg** - Product image for "Midnight Mystery" (400x400 recommended)
5. **perfume4.jpg** - Product image for "Fresh Breeze" (400x400 recommended)

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**: Square format (400x400px) works best for product images
- **Quality**: High-quality, professional product photography
- **Background**: Clean, neutral backgrounds for product shots

## Notes

- The website includes error handling for missing images
- Placeholder text will appear if images are not found
- Images should showcase your perfume bottles clearly
- Consider consistent lighting and styling across all product images

**Replace this file with your actual images when ready to launch!**
